Scrawl-canvas v8 - Physics test 001

Default gravity and drag forces; spring connected particles

Test purpose

  • Create and clone Particle objects
  • Connect Particles together using Spring objects
  • Create entity objects which use Particles as their position pivots
  • Add default Force objects (gravity, air drag) to the Particles
  • Animate the scene, checking that the (Spring and Force mediated) Particles, and entitys, move as expected

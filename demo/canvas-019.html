Test purpose

Drag and drop the entitys to test their sensors correctly collide with underlying entitys

  • Create a set of blocks to act as the grid over which the test entitys will be dragged
  • Create a variety of test entitys of differing types
  • Perform between-group collision checks between test and grid groups, updating colliding grid entitys to show that they are in collision
  • Drag and drop the test entitys across the canvas, to check that test entity sensors are correctly positioned and the between-group collision detection works as expected
  • Vary the collective attributes of the test entitys (handle, offset, rotation, scale, flip) to make sure each entity's sensors correctly account for these changes

Scrawl-canvas v8 - Canvas test 015

Animate gradient by tween

Click on the canvas to start the animation

Test purpose

  • Setup a linear gradient, and a block entity to display it
  • Create a user-defined tween engine function to handle the tween calculations
  • Setup a tween to manipulate the gradient palette's start and end cursors, to be run when the user clicks on the canvas
  • During the animation the gradient should loop 3 times, with an ease covering the length of all loops
  • Check that the tween ignores user clicks while it is running
  • The gradient should remain coherent during the entire tween run, and there should be no flashes of solid black or other solid colors

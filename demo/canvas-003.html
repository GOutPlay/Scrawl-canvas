Canvas controller 'fit' attribute

This canvas is resizable* - drag the bottom right corner to change its dimensions.

Canvas fit

Test purpose

  • Check the various fit modes by which a virtual canvas can be copied into the display canvas

*Note

Actually the stack <div> element containing the <canvas> element is resizable. No browser seems to permit resizable canvases (probably for understandable reasons, though I can't find this prohibition documented anywhere). In fact only Firefox seems to allow the div to be resized: Chrome shows the corner dragger but doesn't respond to user interaction; Edge doesn't even go that far.

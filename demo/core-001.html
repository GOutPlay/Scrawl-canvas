Scrawl-canvas v8 - Demo Core 001

Module loading and initialization

To test that:

  • All Scrawl-canvas modules load into browser
  • All module code successfully runs to completion without error after loading
  • The core animation loop initializes
  • User can add an Animation object to the system
  • Animation object automatically runs
  • User can access scrawl.currentCorePosition object
  • scrawl.currentCorePosition object correctly tracks mouse cursor position
  • scrawl.currentCorePosition object correctly tracks browser/device dimensions
  • scrawl.currentCorePosition object correctly tracks page scroll

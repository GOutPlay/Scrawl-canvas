Test purpose

  • Create a particle world object
  • Add user-defined attributes to the particle world object
  • Create a particle Emitter entity
  • Check to see that the Emitter entity can lock to the mouse cursor
  • Check that particles respond to the gravity force, when applied
  • Check the effect of changing the world object's tickMultiplier attribute
  • Check that particle color, and its opacity effects, can be controlled via the user-defined attributes in the world object
  • Check the effects of changing the following emitter/particle attributes: maxParticles; particleChoke; batchParticlesIn; historyLength; killParticleAfter; range (x, y, z); rangeFrom (x, y, z)
  • Check that the canvas backgroundColor attribute can be updated

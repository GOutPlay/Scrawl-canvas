Scrawl-canvas v8 - Canvas test 012

Cell-locked gradients

Drag and drop the entitys around the canvas

Test purpose

Drag and drop the entitys to test that their gradients change in line with the underlying cell-based gradient (which is shared by all these entitys)

  • Setup a linear gradient
  • Create and clone block and wheel entitys, each using the gradient as a fill
  • The cloned entitys should include a roll delta
  • Check that when the drag operation starts, the gradient does not suddenly jump
  • During the drag operation, the gradient should change in line with the entity's position on the cell (the yellow line should bisect the cell)
  • Check that when the drag operation ends, the gradient does not suddenly jump

Expected results

The gradient, when applied to rotating entitys, will rotate with the entitys. When dragged while rotating, the drop state of the gradient will change according to the degree of entity rotation at the time of the drop.

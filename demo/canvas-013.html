Scrawl-canvas v8 - Canvas test 013

Entity-locked gradients

Drag and drop the entitys around the canvas

Test purpose

Drag and drop the entitys to test that their gradients change in line with the underlying entity-locked gradient (which is shared by all these entitys)

  • Setup a linear gradient
  • Create and clone block and wheel entitys, each using the gradient as a fill; lock the fill to the entity
  • The cloned entitys should include a roll delta
  • The gradient should fit inside each entity, taking into account that entity's scale
  • Check that when the drag operation starts, the gradient remains static within the entity (the yellow line should bisect each entity)
  • Check that when the drag operation ends, the gradient remains static within the entity

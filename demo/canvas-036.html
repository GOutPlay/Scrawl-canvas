Scrawl-canvas v8 - Canvas test 036

Cell functionality (functionality and associated tests needs to be expanded)

This canvas is resizable - drag the bottom right corner to change its dimensions.

Test purpose

Current functionality:

  • Create a range of Cell artefacts with absolute and relative positioning and dimensions
  • Animate the Cells - for instance: delta roll; animate along a path
  • Pivot a Cell to another Cell, and to an entity object (not yet added to demo)
  • Pivot a Cell to mouse cursor (tested separately in Demo DOM-011)

Future functionality (need to check to see if some of these have already been implemented in code, but not exposed in a usable manner):

  • Cell borders, which can: be drawn on the base Cell either before or after the bordered Cell stamps itself onto it; use colors, gradients, patterns for its border; use shadows ('drawAndFill', 'fillAndDraw, 'drawThenFill', 'fillThenDraw')
  • Mouse collision detection - for instance update the Cell (and/or its constituent artefacts) when the mouse hovers over it or the user clicks/taps on it
  • Drag and drop Cells across the Canvas
  • Clone Cells (longer term objective)

Known bugs:

  • The Oval entity does not respond to changes in its Cell's dimensions - check to see if this is true of all Shape-based entitys

Scrawl-canvas v8 - Canvas test 014

Animate gradient by delta values

Test purpose

  • Setup a radial gradient which includes a delta object to manipulate its palette's start end end cursor
  • Setup a block entity to display the gradient
  • During the animation, the center of the gradient should gradually change color as each hoop builds and wanes (in Scrawl-canvas versions prior to v8 this would jump between black and the full color as each color stop moved away from position 0)
  • During the animation, the color beyond the outer radius should gradually change color as each hoop builds and wanes (in Scrawl-canvas versions prior to v8 this would jump between black and the full color as each color stop moved towards position 1)

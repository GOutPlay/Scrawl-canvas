Scrawl-canvas v8 - Canvas test 029

Phrase entitys and gradients

Test purpose

  • Create a linear gradient
  • Create several Phrase objects
  • Position some with relative coordinates
  • Create a Shape entity and position one of the phrases on it with 'path' (delta animated)
  • Check that the gradient appears as expected when it is vertical (top-bottom)
  • Check that the gradient appears as expected when it is horizontal (left-right)
  • Check that the gradient appears as expected for both cell-locked and entity-locked gradients

May 2020 results

Currently experiencing a number of issues and bugs in the mix between phrase and styles code:

  • Cell-locked vertical gradients (seem to) appear as expected.
  • Cell-locked horizontal gradients - the initial letter's gradient appears to be mis-positioned, compared to the other letters in the Phrase.
  • ... For multi-line Phrases this also applies to a few other letters in the text.
  • Entity-locked gradients do not display (they are either set to 0 or 999) - this almost certainly has something to do with Phrase height issues.
  • Phrases locked to a path - all letters appear to use the initial letter's take on the gradient it should be displaying.
  • ... This is a very nice effect with horizontal gradients!
